# Flasp app project for login, secure password and DB

## This is an app that I learned how to create a full functional login page for a user 
it contains a login;
a password with hash (which will encode the user password and make it impossible to decote);
a DB to save and store all the user interface;

## how to use:
Start on the register section, create your name, email and password;
after that you can log in in the "login" section, with your email and passoword only, 
and you will be able to get the secret how many times you want/need
